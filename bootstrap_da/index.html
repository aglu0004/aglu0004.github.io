What is Bootstrap?
Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Bootstrap 5

The upcoming release of Bootstrap will show major changes to the program. Such as Dropping jQuery for vanilla JavaScript, rewriting the grid to support columns that are outside of rows and responsive gutters, and also adding a custom set of SVG icons

Mobile First

Bootstrap includes a powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, five default responsive tiers, Sass variables and mixins, and dozens of predefined classes.

Bootstrap Color

Bootstrap has a pre-set list of color groups that may be used on elements such as text, background, and border. These color groups are; Primary, Secondary, Success, Danger, Warning, Info, Light, Dark, and Muted

Alerts

Alerts are a great way to stylize your Bootstrap page, with these you can provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.
